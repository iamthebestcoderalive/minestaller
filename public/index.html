minestaller // welcome
Select your connection method:

• Use Agent (Recommended): Connects via a lightweight local companion. Best for standard installations, as it bypasses browser sandbox blocks on default folders like AppData.

• Connect Folder (No Agent): Links a directory directly. Note: Browsers block selecting folders inside AppData (used by the standard launcher and Modrinth App) due to security rules. Use this option if your game directory is set to a custom folder outside AppData (e.g. in Documents, Desktop, or another drive).
